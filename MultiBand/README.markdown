# MultiBand

Three-band compressor with adjustable crossover frequencies. Each band has independent compression and output level controls with shared attack and release times. Includes a listen mode for soloing individual bands, stereo width control, and mid/side processing option.

| Parameter | Description |
| --------- | ----------- |
| Listen | Solo a frequency band for monitoring: Low, Mid, High, or Output (all bands) |
| L <> M | Crossover frequency between the low and mid bands |
| M <> H | Crossover frequency between the mid and high bands |
| L Comp | Compression amount for the low band |
| M Comp | Compression amount for the mid band |
| H Comp | Compression amount for the high band |
| L Out | Output level for the low band |
| M Out | Output level for the mid band |
| H Out | Output level for the high band |
| Attack | Compressor attack time (shared across all bands, scaled per band) |
| Release | Compressor release time (shared across all bands, scaled per band) |
| Stereo | Stereo width preservation |
| Process | Processing mode: M (mid/side disabled) or S (mid/side enabled) |
