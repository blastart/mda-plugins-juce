# ThruZero

Through-zero flanger. Unlike a standard flanger that only delays the signal, this effect sweeps the delay time through zero and into negative territory (by delaying the dry signal instead), creating the authentic sound of two tape machines with varying speed. Produces classic jet-plane flanging with the characteristic "null" at zero delay.

| Parameter | Description |
| --------- | ----------- |
| Rate | LFO rate for the flange sweep (displays cycle period in seconds) |
| Depth | Depth of the delay modulation |
| Mix | Wet / dry mix - set to 50% for classic flanging |
| Feedback | Amount of feedback for a more resonant, metallic sound |
| DepthMod | Modulates the depth offset - controls the shape of the sweep |
