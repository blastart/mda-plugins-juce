# Transient

Attempt to manipulate the attack and release characteristics of audio independently. Uses two pairs of envelope followers with different time constants to separate the transient (attack) and sustain (release) components of the signal, then applies gain modulation to emphasize or suppress each. Particularly useful on drums and percussive material.

| Parameter | Description |
| --------- | ----------- |
| Attack | Transient (attack) emphasis - positive values boost attack, negative values soften it |
| Release | Sustain (release) emphasis - positive values boost sustain, negative values reduce it |
| Output | Level trim |
| Filter | Detection filter - selects low or high frequency focus for the envelope followers |
| Att Hold | Attack envelope hold time |
| Rel Hold | Release envelope hold time |
