# Combo

Amp and speaker simulator. Attempt to recreate the sound of various amplifier and speaker cabinets using a combination of clipping (soft or hard), comb filtering, and multi-pole low-pass / high-pass filtering. Can also be used for adding "warmth" to digital recordings or processing stereo signals independently.

| Parameter | Description |
| --------- | ----------- |
| Model | Speaker/cabinet model: D.I., Speaker Sim, Radio, Mesa Boogie 1", Mesa Boogie 8", Marshall 4x12 (on-axis), Marshall 4x12 (off-axis) |
| Drive | Distortion amount - left half is soft clipping, right half is hard clipping |
| Bias | Asymmetry of the clipping - can add even harmonics |
| Output | Level trim |
| Process | Processing mode: Mono (sums input) or Stereo (processes channels independently) |
| HPF Freq | High-pass filter frequency for pre-distortion tone shaping |
| HPF Reso | High-pass filter resonance |
