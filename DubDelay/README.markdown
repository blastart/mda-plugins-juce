# DubDelay

Delay effect with feedback saturation or limiting, feedback tone control, and LFO modulation. Designed for creating "dub" style echoes with character. The feedback path includes a low-pass/high-pass filter and a limiter or saturator to keep repeats under control while adding color.

| Parameter | Description |
| --------- | ----------- |
| Delay | Delay time |
| Feedback | Feedback amount - left half uses saturation, right half uses limiting |
| Fb Tone | Feedback filter tone - low-pass to left, high-pass to right |
| LFO Dep. | LFO modulation depth for chorus/vibrato effect on the delay |
| LFO Rate | LFO modulation rate |
| FX Mix | Wet / dry mix |
| Output | Level trim |
