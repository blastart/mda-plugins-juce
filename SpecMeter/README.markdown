# SpecMeter

Multi-band spectrum analyzer and level meter. Passes audio through unmodified while measuring peak levels, RMS levels, stereo correlation, and per-octave spectral energy using a polyphase filter bank. Provides up to 13 octave bands of spectral analysis. This is a metering/analysis tool, not an audio effect.

| Parameter | Description |
| --------- | ----------- |
| Gain | Input gain for metering |
| RMS Speed | Smoothing speed for RMS level measurement |
| Spec Speed | Decay speed for spectrum display |
