# RePsycho

Triggered pitch-shifting effect. When the input signal exceeds the threshold, a section of audio is captured and played back at a different pitch. Creates "stutter" and pitch-shifting effects on drums and percussive material. Can transpose up or down by up to two octaves with adjustable decay envelope and hold time.

| Parameter | Description |
| --------- | ----------- |
| Tune | Pitch shift amount in semitones |
| Fine | Fine pitch adjustment in cents |
| Decay | Amplitude envelope decay - controls whether the shifted sound fades in or out |
| Thresh | Input level threshold for triggering playback |
| Hold | Minimum time between triggers |
| Mix | Wet / dry mix |
| Quality | Processing quality: Low (mono, no interpolation) or High (stereo with interpolation) |
