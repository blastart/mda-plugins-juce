# RoundPan

3D panning effect using a simple equal-power panning law. Sums the input to mono and pans it around the stereo field. Includes an auto-rotation mode for automatic circular panning at adjustable speed and direction.

| Parameter | Description |
| --------- | ----------- |
| Pan | Pan position in degrees |
| Auto | Automatic rotation speed and direction - center is stopped, left of center rotates one way, right of center rotates the other |
