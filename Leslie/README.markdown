# Leslie

Rotary speaker simulator. Attempt to recreate the sound of a Leslie cabinet, with independently controllable high and low frequency rotors producing both amplitude modulation (throb) and Doppler pitch modulation (width/depth). A crossover filter separates the signal into low and high bands, each processed by its own virtual rotor.

| Parameter | Description |
| --------- | ----------- |
| Speed Mode | Rotor speed: Stop, Slow, or Fast |
| Lo Width | Stereo spread of the low frequency rotor |
| Lo Throb | Amplitude modulation depth of the low frequency rotor |
| Hi Width | Stereo spread of the high frequency rotor |
| Hi Depth | Doppler pitch modulation depth of the high frequency rotor |
| Hi Throb | Amplitude modulation depth of the high frequency rotor |
| Crossover | Crossover frequency between low and high rotors |
| Output | Level trim |
| Speed | Overall speed multiplier for both rotors |
