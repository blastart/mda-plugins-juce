# DeEss

Simple de-esser. Uses a 2nd-order crossover filter to isolate high frequency energy and applies dynamic gain reduction when sibilance exceeds the threshold. Can also be used as a high frequency limiter or to tame overly bright recordings.

| Parameter | Description |
| --------- | ----------- |
| Thresh | Detection threshold |
| Freq | High frequency crossover point for sibilance detection |
| HF Drive | Additional high frequency gain, for fine-tuning sensitivity |
